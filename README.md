# Gen-Ai-app
Gen-AI Android App

Gen-AI is an Android application that harnesses the power of Google’s Gemini API to provide intelligent, generative AI responses. Whether you're crafting content, asking questions, or exploring ideas, Gen-AI delivers fast and context-aware outputs on your mobile device.


---

Features

Gemini API Integration – Powered by Google's large language model for high-quality responses.

Prompt-Based Generation – Input any query or task and get intelligent output.

Minimal UI/UX – Clean, responsive, and mobile-first interface.

Fast and Lightweight – Optimized for performance across Android devices.

Clipboard Support – Easily copy AI-generated content.

Getting Started

Prerequisites

Android Studio (latest)

A valid Gemini API Key

Internet permission in AndroidManifest.xml:


<uses-permission android:name="android.permission.INTERNET" />

Installation

1. Clone the repository:



git clone https://github.com/Ashish11828/gen-ai-app.git

2. Open the project in Android Studio.


3. Add your API key to local.properties:



GEMINI_API_KEY=your_gemini_api_key_here

4. Build and run the app on an emulator or physical device.




---

Usage

1. Launch the app


2. Type your prompt


3. Tap on Generate


4. View and copy the AI-generated output
